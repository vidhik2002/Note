# Note
A website that helps create, edit and delete notes with features like search bar and themes, using React!
#### Pre-requisite
- [React](https://reactjs.org/)
- [npm](https://www.npmjs.com/)

### Installation
 
1. Clone the repo
```sh
git clone https://github.com/vidhik2002/Note.git
```
2. Run the command
```sh
npm start
```
